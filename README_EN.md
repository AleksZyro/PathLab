# PathLab

[Deutsch](./README.md) | **English**

PathLab is an interactive pathfinding visualizer for learning and comparing search algorithms on a weighted grid.

## Status

Stable portfolio version.

## Features

- BFS, DFS, Dijkstra, and A*
- walls and weighted terrain
- animated visualization and algorithm comparison
- undo and redo
- German and English content
- light and dark mode
- automated tests with Vitest

## Tech Stack

React, JavaScript, Vite, CSS, and Vitest.

## Setup

```bash
npm install
npm run dev
```

## Tests

```bash
npm test
```
